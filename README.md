# vibe
VR multiplayer game

WT: Eigenlicht 'Intrinsic Light'

CONTROLS:
- Click anywhere to enable mouse look
- Press ESC to exit mouse look (not fullscreen)
- Press F to toggle fullscreen mode
- WASD/Arrow keys to move
- Shift to toggle run
